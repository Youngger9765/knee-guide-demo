# 膝關節置換術衛教指南 Demo

義大醫院參訪專用 - 人工膝關節置換術衛教指南互動網頁

## 🎯 功能特色

- 📖 完整的膝關節置換術衛教內容
- 🤖 智慧 Chatbot 問答系統（35+ 問答對）
- 🌏 多語言關鍵字支援（繁體/簡體/英文）
- 📱 響應式設計（支援手機/平板/桌機）
- ⚡ 純前端運作（無需後端伺服器）
